# Quick Start Guide - Streamlit Dashboard

Get started with the LBO Model Generator Streamlit dashboard in 5 minutes.

## Step 1: Install Dependencies

```bash
pip install -r requirements.txt
```

## Step 2: Launch Dashboard

```bash
streamlit run app.py
```

The dashboard will open automatically in your browser at `http://localhost:8501`

## Step 3: Load a Test Case (Optional)

1. Navigate to **Assumptions** page (via sidebar)
2. Select a test case from the dropdown:
   - AlphaCo
   - DataCore
   - SentinelGuard
   - VectorServe
3. Click **"Calculate LBO Model"**

## Step 4: View Results

Navigate to **Dashboard** page to see:
- Equity IRR and MOIC
- Exit metrics
- Financial statement trends

## Step 5: Explore Analysis

Go to **Analysis** page for:
- Sensitivity analysis
- Scenario comparison
- Enhanced visualizations
